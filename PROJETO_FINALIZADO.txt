🚀 COMO FAZER DEPLOY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PASSO 1: Git Local (5 min)
  $ git init
  $ git add .
  $ git commit -m "feat: MB Hub v2.0 pronto para produção"

PASSO 2: GitHub (5 min)
  1. Acesse: https://github.com/new
  2. Nome: mb-hub-crm
  3. Crie o repositório

PASSO 3: Conectar Git (5 min)
  $ git remote add origin https://github.com/SEU-USER/mb-hub-crm.git
  $ git branch -M main
  $ git push -u origin main

PASSO 4: Deploy Vercel (5 min)
  1. Acesse: https://vercel.com/new
  2. Conecte GitHub
  3. Selecione mb-hub-crm
  4. Configure environment variables
  5. Deploy!

PASSO 5: Teste (10 min)
  1. Acesse: https://mb-hub-crm.vercel.app
  2. Login: marcia.braga@mbsolucoes.com.br
  3. Teste em mobile (iPhone + Android)
  4. Verificar performance
